# 국내 카드사 고객 이탈 예측 및 마케팅 전략 제언 시스템 (ChurnSight)

> 고객 이탈을 사후 대응이 아닌 **사전 예측 + 즉시 실행**으로 전환하는  
> 머신러닝 × LLM 기반 카드사 CRM 마케팅 의사결정 지원 시스템

---

## 1. 프로젝트 개요

- **프로젝트명**: 국내 카드사 고객 이탈 예측 및 마케팅 전략 제언 시스템
- **프로젝트 성격**: 머신러닝 기반 데이터 분석 & 마케팅 전략 자동화 MVP
- **기간**: 2025.12
- **팀명**: “일단 해보조 (Just Do It)”
- **목표**
  - 고객별 이탈 가능성을 사전에 예측하고
  - 이탈 위험도에 따라 고객을 세분화하며
  - 고객 단위 맞춤형 마케팅 전략을 자동으로 제언하는 실행 중심 시스템 구축

---

## 2. 프로젝트 추진 배경

국내 카드사는 다음과 같은 구조적 문제에 직면해 있음.

- 가맹점 수수료 규제 및 고금리로 인한 **수익성 악화**
- 핀테크·빅테크 중심의 결제 플랫폼 확장으로 **고객 접점 상실**
- 신규 고객 확보 비용 증가 대비 **기존 고객 수익 기여도 감소**

특히 카드사 고객 이탈은  
단일 원인이 아닌 **혜택·비용·경험·경쟁 전환 등 복합적 요인**으로 발생하며,  
기존의 사후 마케팅 방식으로는 효율적인 대응이 어려움.

이에 따라 **이탈을 사전에 예측하고, 선제적으로 개입할 수 있는 데이터 기반 체계**가 필요함.

---

## 3. 문제 정의 (AS-IS / TO-BE)

### AS-IS
- 고객 이탈을 **이용 중단 이후에 인지**
- 일괄 캠페인 중심의 마케팅 운영
- 데이터는 존재하지만 **리포팅·사후 분석에만 활용**
- 이탈 관리 KPI가 후행 지표 중심

### TO-BE
- 고객별 **이탈 확률 사전 예측**
- 위험군 기반 차별화된 마케팅 전략 적용
- 분석 결과가 즉시 실행으로 연결되는 구조
- 이탈 예측을 **선행 KPI**로 활용

고객 이탈을 “사후 관리 대상”이 아닌  
**“사전 예측 · 전략 제언 대상”으로 전환**

---

## 4. 해결 방안 개요

본 프로젝트는 다음 흐름의 End-to-End 시스템을 구축함.

1. 고객 거래·행동·CX 데이터 기반 이탈 확률 예측
2. 예측 확률에 따른 위험군(Tier) 분류
3. 위험군 및 고객 특성에 맞는 마케팅 전략 자동 도출
4. Streamlit UI를 통해 실무자가 즉시 활용 가능하도록 시각화

---

## 5. 시스템 아키텍처 (MVP)

- **Machine Learning**: LightGBM 기반 고객 이탈 확률 예측
- **LLM**: GPT API를 활용한 고객 단위 마케팅 전략 생성
- **Frontend / App**: Streamlit 기반 대시보드

### 데이터 흐름 요약

CSV 업로드
1. 전처리 & Feature Engineering
2. 이탈 확률 예측 (LightGBM)
3. 위험군 분류 (Threshold 기반)
4. 고객 선택
5. LLM 전략 생성
6. UI 카드/표로 시각화

---

## 6. 주요 기능

### 6.1 로그인 페이지
- 내부 사용자 접근 통제를 위한 간단 인증

### 6.2 데이터 입력 페이지
- 고객 데이터 CSV 업로드
- 분석 실행을 통해 이탈 예측 결과 생성

### 6.3 이탈 가능 고객 추출 페이지
- 이탈 위험군별 고객 리스트 제공
- 이탈 확률 기준 정렬 및 상세 정보 확인

### 6.4 마케팅 전략 제언 페이지
- 고객 1명 선택 시
- LLM 기반 초개인화 마케팅 전략 자동 생성
- 추천 채널, 오퍼, 메시지 방향을 UI로 제공

## 7. 머신러닝 모델링

### 모델 선정
- **LightGBM**
  - 불균형 데이터 환경에서 Recall 성능 우수
  - 확률 예측 기반 위험도 활용에 적합
  - 학습 속도 및 튜닝 효율성 우수

### 평가 기준
- ROC-AUC
- Recall (이탈자 미포착 최소화)
- Accuracy는 클래스 불균형으로 제외

### 핵심 설계 포인트
- 이탈 여부 이진 판단 
- **이탈 확률 기반 위험군 관리

---

## 8. 이탈 위험군 정의 및 전략

| 위험군 | 의미 | 전략 방향 |
|------|------|---------|
| 즉시 이탈 위험 | 이탈 임박 | 방어 |
| 고위험군 | 부정 경험 누적 | 방어 |
| 중위험군 | 사용 패턴 약화 | 선제 |
| 저위험/안정군 | 관계 안정 | 활성 |

### 전략 프레임
- **방어**: 불만·장애·혜택 이슈 즉각 해결
- **선제**: 이탈 조짐 고객 사전 리텐션
- **활성**: 사용 습관 회복 및 관계 강화

---

## 9. LLM 기반 마케팅 전략 고도화

LLM(GPT)은 단순 설명 도구가 아닌  
**마케팅 실행을 가속하는 의사결정 보조 도구**로 활용됨.

### 활용 방식
- 고객 행동·이탈 맥락 요약
- 고객 단위 맞춤 전략 자동 생성
- 실무자가 바로 활용 가능한 액션 가이드 제공

### 기대 효과
- 무차별 할인 감소
- 초개인화 마케팅 실행력 강화
- 분석 → 실행 간 의사결정 시간 단축

---

## 10. 기대 효과

- **비용 효율적 이탈 관리**
  - 신규 고객 확보 대비 낮은 비용으로 수익성 개선
- **마케팅 실행 속도 향상**
  - 분석 결과를 즉시 액션으로 연결
- **데이터 기반 개인화 체계 구축**
  - 고객을 집단이 아닌 개인 단위로 관리
- **확장 가능성 확보**
  - 실시간 데이터·자동 캠페인·성과 대시보드로 확장 가능
